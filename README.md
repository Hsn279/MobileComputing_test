# 📱 Mobile Computing - Aplikasi Mobile Testing

Selamat datang di repositori proyek **Mobile Computing**! Repositori ini dibuat sebagai wadah pembelajaran, eksperimen, dan pengujian dalam pengembangan aplikasi mobile selama menempuh mata kuliah Mobile Computing. 

Proyek ini mencakup seluruh proses pengembangan, mulai dari perancangan antarmuka (UI/UX) hingga implementasi kode sumber (*source code*) dan tahap pengujian (*testing*).

---

## 🚀 Fitur Utama & Cakupan Testing
* **UI/UX Implementation:** Mengubah desain *high-fidelity* menjadi komponen aplikasi yang responsif.
* **State Management:** Pengujian alur data dan logika aplikasi.
* **API Integration (jika ada):** Pengujian konektivitas data eksternal.
* **Unit & Integration Testing:** Memastikan setiap fungsi dan halaman berjalan dengan modul yang sesuai tanpa *error*.
